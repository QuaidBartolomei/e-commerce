## Getting Started

### Prerequisites

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

2. Clone the repo
   ```sh
   git clone https://github.com/quaidbartolomei/fine-attire.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Add your firebase credentials to a `.env` file based on `.env.template`
