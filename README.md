## Getting Started

### Prerequisites

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/quaidbartolomei/fine-attire.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Add your firebase credentials to a `.env` file based on `.env.template`

<!-- USAGE EXAMPLES -->

## Usage

Run the dev server:

```sh
npm run dev
```
